# 📊 Shopping Data Analysis (Power BI)

## 🔍 Overview
This project focuses on analyzing shopping data to understand customer behavior, purchasing patterns, and sales performance using Power BI. The dashboard provides meaningful insights to support data-driven decision-making.

## ⚙️ Tools & Technologies
- Power BI
- Microsoft Excel

## 📈 Key Features
- Developed an interactive dashboard to analyze customer behavior  
- Visualized sales trends and purchasing patterns  
- Identified key customer segments and buying habits  
- Implemented filters and slicers for dynamic data exploration  
- Performed data cleaning and transformation for accurate analysis  

## 📊 Key Insights
- Identified customer purchasing trends and preferences  
- Analyzed product performance and sales patterns  
- Highlighted important factors influencing customer behavior  
- Provided insights to improve business strategies
